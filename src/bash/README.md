# bash

- a bash script starts with a shebang

```bash
#!/bin/bash
```

- adding comments

```bash
# comments
```

- a bash script usually contains commands one below the other. One can `;` to run multiple commands on same line

```bash
ls; pwd
```

- a bash script needs to be an executable to run

```bash
chmod +x ./main.sh
```

```bash
./main.sh
```

- or, can directly use `bash` cmd to run it

```bash
bash main.sh
```
